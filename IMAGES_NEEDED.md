## 🎨 IMAGE STYLE GUIDELINES

### Photography Style:
- **Authentic:** Real, genuine moments (not overly staged)
- **Diverse:** Show diversity in age, ethnicity, and background
- **Warm:** Warm color tones, natural lighting preferred
- **Professional:** Clean, high-quality images
- **Respectful:** Dignified portrayal of seniors and care situations

### Color Palette to Match:
- Deep Red: #C62828 (accents, CTAs)
- Dark Navy: #0F172A (text, dark sections)
- Soft Blue: #1E3A8A (trust elements)
- Light Gray: #F8FAFC (backgrounds)

### People Representation:
- Caregivers: Professional, caring, diverse ages (25-55)
- Seniors: Dignified, happy, diverse (65+)
- Families: Supportive, grateful, diverse

## 📥 IMAGE SOURCES RECOMMENDATIONS

### Stock Photo Sites (Royalty-Free):
1. **Unsplash** (free) - unsplash.com
   - Search: "elderly care", "caregiver", "senior homecare", "compassionate care"

2. **Pexels** (free) - pexels.com
   - Search: "homecare", "caregiver senior", "elderly assistance"

3. **Shutterstock** (paid) - shutterstock.com
   - Search: "home health aide", "senior care", "in-home caregiver"

4. **iStock** (paid) - istockphoto.com
   - Search: "elderly care at home", "professional caregiver"

### AI Image Generation (if needed):
- **Midjourney** or **DALL-E** can generate custom images
- Prompt example: "Professional caregiver assisting elderly person at home, warm lighting, caring expression, diverse, photorealistic, editorial photography style"

## 💡 TIPS

1. **Optimize for Web:** Compress images using TinyPNG or Squoosh before uploading
2. **Consistent Style:** Try to use images from the same photographer/series for consistency
3. **Alt Text:** All images have descriptive alt text already in the code for accessibility
4. **Fallback:** The website has fallback styling if images are missing, but they greatly improve the professional appearance

---

## 🔧 AFTER UPLOADING

Once you've uploaded all images to `/public/images/`, run:
```bash
npm run build
```

To verify everything works correctly.
